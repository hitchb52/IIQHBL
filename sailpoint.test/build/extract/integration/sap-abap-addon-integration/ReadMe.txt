SailPoint's SAP ABAP Add-On Integration
=======================================
The "sap-abap-addon-integration" folder delivers a "SAILPOIN-100_2022.zip" file containing the "PAT" files required for installing SailPoint's ABAP Add-On integration named "SAILPOIN" and, importing its support packages.
This integration is an SAP-certified integration provided by SailPoint to replace the use of SAPâ€™s RFC_READ_TABLE function module - according to SAP recommendations - for reading data from SAP tables required for governance.
The integration is designed to provide enhanced security, support for pagination and performance improvements over RFC_READ_TABLE.
Please refer SAP GRC, SAP Direct Connector's documentation for detailed information about the PAT files, Add-On installation/un-installation, usage, configuration and benefits of this integration.
